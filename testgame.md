# limitwalk

```
ランダム生成ダンジョン。
２０＊２０マス。
０階層はタイトル。
１階層からスタート。
昇り階段は３の倍数にしかない。スコアを精算する事ができる。
階段を降りる度に、生命力が１００、回復する。
一歩歩く毎に、生命力が１減る。５階層を超えると、２減る。１０階層を超えると３減る。
毒の床を歩く毎に、生命力が１０減る。
宝箱は三択で、生命力をX回復、生命力をX減らしスコアY倍化、歩行数をX加算、のいずれか。
ｘは１０から１００、Yは１から５
スコアは、階層＊歩行数を一歩毎に加算する。
３の倍数層は、地上に戻る階段がある。地上に戻れば、帰還時の階層＊現在のスコアを最終スコアとする。
生命力が尽きれば、現在のスコアを最終スコアとする。
最大は、１９階層。

スコア、歩行数、開けていない宝箱の数は情報開示する。宝箱の数は０から４

スコアのリストはトップ１０まで。

```
